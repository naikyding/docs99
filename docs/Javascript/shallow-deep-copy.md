# Shallow VS Deep Copy 淺拷貝與深拷貝

:::tip 簡單說
**淺拷貝** 只是 `傳址` (by reference) 或 拷貝到第一層；
**深拷貝** 拷貝的目標將會是獨立的個體，不會與來源有任何連動。
:::

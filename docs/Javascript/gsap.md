# GSAP 動畫特效

## 說明

GSAP v3 統一使用 `gsap` 物件，做為操作的根本，使用其來製作所有動畫特效。

```js
// old
TweenMax.to('.box', 1, { x: 100 })

// gsap3
gsap.to('.box', {
  duration: 1,
  x: 100,
})
```
